【系统常见错误码与排查排障 Runbook】

错误码：ERR-GW-001
现象：网关返回 401 Unauthorized，TraceId 丢失。
原因与排查步骤：通常是因为请求头中未携带有效的 Token，或者 Token 已过期。请检查前端请求头，并确认网关 AuthFilter 是否正常加载。

错误码：ERR-RPC-503
现象：调度中心触发任务时，RPC 客户端抛出 "No Provider available" 异常。
原因与排查步骤：找不到可用的远程服务提供者。排查步骤：1. 检查 ZooKeeper 中该服务节点是否在线。 2. 检查业务服务是否因 OOM 或宕机掉线。 3. 检查本地 Watcher 缓存是否未及时刷新。

错误码：ERR-SCHED-002
现象：任务一直处于 RUNNING 状态无法结束（死锁）。
原因与排查步骤：软超时与硬超时机制失效。排查步骤：进入调度中心数据库，检查 JobInstance 的软超时时间设置，手动触发软超时扫描线程，并使用 CAS 机制将状态强制置为 TIMEOUT。
